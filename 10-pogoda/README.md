# Раздел 10: Погода — Данные из интернета

Это поворотный раздел. Впервые твоё приложение будет получать реальные данные из интернета — настоящую погоду в настоящем городе. Ты узнаешь, что такое API и как с ним работать.

---

## Проект: Погодное приложение

Текущая погода по геолокации, поиск по городу, прогноз на 5 дней, иконки погоды, pull-to-refresh.
