# Flutter с нуля до Junior — Полный курс

Добро пожаловать! Этот курс проведёт тебя от полного нуля до уровня Junior Flutter-разработчика. Никакого предыдущего опыта в программировании не нужно.

За курс ты построишь **8 реальных приложений** своими руками — от простой визитки до полноценного трекера финансов с графиками и локальным хранением данных.

---

## Что ты получишь

- **135 уроков** (~19 часов видео) — короткие и по делу
- **8 проектов** с пошаговым разбором — каждый сложнее предыдущего
- **4 самостоятельных проекта** — проверишь себя без подсказок
- **Баг-хантинг сессии** — научишься находить и чинить ошибки
- **Разминки** в начале каждого раздела — чтобы знания не забывались

---

## Как проходить курс

1. **Смотри уроки по порядку** — каждый следующий строится на предыдущем
2. **Пиши код вместе с видео** — не просто смотри, а повторяй руками
3. **Делай челленджи** — попробуй сам, прежде чем смотреть решение
4. **Не пропускай разминки** — они помогают вспомнить и закрепить
5. **Лучше 20 минут каждый день**, чем 3 часа в субботу

```
Визитка           → Первое приложение на экране — магия!
Профиль           → Красивая вёрстка, как в настоящих приложениях
Генератор цитат   → Приложение реагирует на нажатия
🏆 КНБ            → Первый проект полностью сам
Конвертер валют    → Два экрана, формы, ввод данных
Рецепты           → Вкладки, списки, тёмная тема
🏆 Помодоро       → Второй проект сам — уже сложнее
Погода            → Данные из интернета в реальном времени
Чат               → Регистрация, Firebase, сообщения
🏆 Заметки        → Третий самостоятельный — уже уверенно
Задачи            → Профессиональное управление состоянием
Хранение          → Данные живут после перезапуска
🏆 Трекер финансов → Финал: ВСЁ вместе в одном приложении
```

## Что ты будешь уметь после курса

- Создавать мобильные приложения на Flutter для Android и iOS
- Работать с API и получать данные из интернета
- Подключать Firebase (авторизация, база данных, реальное время)
- Управлять состоянием приложения с помощью Provider
- Сохранять данные локально (Hive, SharedPreferences)
- Делать красивый UI с анимациями и тёмной темой
- **Портфолио из 10+ приложений** для поиска работы
