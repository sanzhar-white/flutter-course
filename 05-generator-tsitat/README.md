# Раздел 5: Генератор цитат — Приложение оживает

До этого экраны были статичными. Теперь приложение будет реагировать на действия — нажал кнопку, и всё изменилось. Здесь же ты впервые познакомишься с языком Dart.

## Проект: Генератор мотивационных цитат

Нажимаешь кнопку → появляется случайная цитата → меняется цвет фона. Есть кнопка «Поделиться», чтобы отправить цитату друзьям.

## Почему Dart появляется только сейчас?

Мы не учим Dart отдельно — это скучно и бессмысленно. Вместо этого ты изучаешь Dart когда он нужен для проекта. Переменные — потому что хранишь цитату. Списки — потому что цитат много. Функции — потому что нужно выбрать случайную. Так знания сразу привязываются к практике.
