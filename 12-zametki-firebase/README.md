# Раздел 12: Заметки — Третий самостоятельный проект

Ты уже знаешь Firebase, авторизацию и работу с облачной базой данных. Пора использовать всё это самостоятельно.

---

## Твоё задание

Создай приложение «Заметки» — пользователь регистрируется, создаёт заметки, редактирует и удаляет их. Все данные хранятся в Firebase.

## Что тебе понадобится

Всё, что ты узнал в разделах 10–11:
- Firebase Auth (регистрация и вход)
- Cloud Firestore (CRUD — создание, чтение, обновление, удаление)
- Навигация между экранами
- Красивая вёрстка списка

> Если чувствуешь, что забыл что-то из Firebase — это нормально. Вернись в раздел 11, пересмотри нужный урок, и продолжай. Повторение — часть процесса.
