# Раздел 8: Рецепты — Списки и красивый дизайн

Ты построишь красивое приложение с рецептами — списки с картинками, экран деталей, вкладки внизу. Впервые сделаешь тёмную тему. Это приложение уже будет выглядеть профессионально.

---

## Проект: «Мои рецепты»

Три вкладки: Все рецепты / Избранное / Профиль. Нажимаешь на рецепт — открывается красивый экран с картинкой, ингредиентами и шагами приготовления.
